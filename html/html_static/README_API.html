How to use

Send "GET" or "POST" request to "http://DOMEN/api/<method>?[args]" or "http://DOMEN/<lang>/api/<method>?[args]"

Methods

  • server.getPublicKey
    Return public key for encrypt password

  • account.login
    Return token for using api
    • <login> - User`s login
    • <password> - User`s password encrypted public key
    • [remember] - If true, request return unlimit token

  • account.get
    Return information about user
    • <id> - User`s id (If -1 return information about user self)
    • <token> - User`s token

  • account.statusWith
    Return status with user.
    • <id> - User`s id
    • <token> - User`s token


    Status
    1. friend - This user is friend
    2. sendRequest - You send friends request to this user
    3. getRequest - You get friends request from this user
    4. null - You are never have relations with this user
    5. none- You have not relations with this user
    6. in_black - You added this user to black list
    7. black - This user added you to black list
    8. all_black - This user added you to black list and you added him to black list

  • account.edit
    Edit user
    • <input> - JSON object about edit information
    • <token> - User`s token

  • blacklist.add
    Adding user to black list if it can
    • <id> - User`s id
    • <token> - User`s token

  • blacklist.delete
    Deleting user from black list if it can
    • <id> - User`s id
    • <token> - User`s token

  • blacklist.get
    Return array of id who you added to black list
    • <token> - User`s token

  • friends.add
    Send request to user or accept friend`s request
    • <id> - User`s id
    • <token> - User`s token

  • friends.delete
    Deleting user from friends if it can or delete sended request
    • <id> - User`s id
    • <token> - User`s token

  • friends.get
    Return array of id who added to friends user
    • <id> - User`s id (If -1 return information about user self)
    • <token> - User`s token

  • friends.getSendRequest
    Return array of id who you send friend`s request
    • <token> - User`s token

  • friends.getGetRequest
    Return array of id who send friend`s request to you
    • <token> - User`s token

  • photos.add
    ONLY POST REQUEST!!!
    Upload photo to server
    • <photo> - Photo (in body)
    • <access> - Access of photo
    • <token> - User`s token


    Accesses
    • 0 - Only user who upload image
    • 1 - Only friends of user who upload image
    • 2 - Only friends of friends of user who upload image
    • 3 - All

  • photos.get
    Return file of photo
    • <id> - Photo`s id
    • <type> - Type of photo
    • <token> - User`s token
    Type of photos
    • s - Max size of side: 75px
    • m - Max size of side: 130px
    • x - Max size of side: 604px
    • o - if width/height <= 1.5 return image with max width 130px else return crop image with aspect = 1.5 and max width = 130px
    • p - if width/height <= 1.5 return image with max width 200px else return crop image with aspect = 1.5 and max width = 200px
    • q - if width/height <= 1.5 return image with max width 320px else return crop image with aspect = 1.5 and max width = 320px
    • r - if width/height <= 1.5 return image with max width 510px else return crop image with aspect = 1.5 and max width = 510px
    • y - Max size of side: 807px
    • z - Max size: 1080x1024
    • w - Max size: 2560x2048

  • photos.setTarget
    Set value
    You don`t write type, because type for avatar`s is 2.
    • <id> - Photo`s id
    • <target> - Photo`s target
    • <token> - User`s token
    Targets
    • 1 - User`s avatar. Warning: photo will be available for all users

  • photos.getTarget
    Return value
    You don`t write type, because type for avatar`s is 2.
    • <id> - User`s id (If -1 return information about user self)
    • <target> - Photo`s target
    • <token> - User`s token
