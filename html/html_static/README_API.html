How to use

Send "GET" or "POST" request to "http://DOMEN/api/<method>?[args]" or "http://DOMEN/<lang>/api/<method>?[args]"

Methods

  • account.login
    Return token for using api
    • <login> - User`s login
    • <password> - User`s password
    • [remember] - If true, request return unlimit token

  • account.ban
    Return true if user was add to black list
    • <id> - User`s id who need delete from black list
    • <token> - User`s token

  • account.get
    Return information about user
    • <id> - User`s id
    • <token> - User`s token

  • account.getSelf
    Return information about user self
    • <token> - User`s token

  • account.changePassword
    For change password
    • <password> - User`s old password
    • <newpassword> - User`s new password
    • <token> - User`s token

  • account.getBanned
    Return array of users` id who is in black list
    • <token> - User`s token

  • account.getFriends
    Return array of users` id who is friends
    • <token> - User`s token

  • account.unBan
    Return true if user`s was delete from black list
    • <token> - User`s token

  • account.statusWith
    Return status with user.
    • <id> - User`s id
    • <token> - User`s token


    Status
    1. friend - This user is friend
    2. sendRequest - You send friends request to this user
    3. getRequest - You get friends request from this user
    4. none or null or undefined - You have not relations with this user
    5. in_black - You added this user to black list
    6. black - This user added you to black list

  • friends.add
    Adding user to friends if it can
    • <id> - User`s id
    • <token> - User`s token

  • friends.delete
    Deleting user from friends if it can
    • <id> - User`s id
    • <token> - User`s token

    • Work in progress. Methods can change or have errors

  • photos.add
    ONLY POST REQUEST!!!
    Download photo to server
    • <photo> - Photo (in body)
    • <type> - Type of photo
    • <token> - User`s token


    Type of photos
    1. Very small. Max size 50x50
    2. Medium. Max size 200x300
    3. Big. Max size 1920x1080

  • photos.getURL
    Get url for download photo
    • <id> - Photo`s id
    • <type> - Type of photo
    • <token> - User`s token

  • photos.setAvatar
    Adding user to friends if it can
    You don`t write type, because type for avatar`s is 2.
    • <id> - Photo`s id
    • <token> - User`s token
